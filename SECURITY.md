# Security Policy

## Reporting a vulnerability

If you discover a potential security issue related to the Janus Governance Core specifications or reference implementations, please report it responsibly.

Contact the project maintainers through the repository issue tracker or by private communication if the issue requires discretion.

## Scope

Security issues may include:

- vulnerabilities in governance event logging
- tampering risks in append-only audit logs
- weaknesses in evidence traceability
- risks in governance evaluation mechanisms

## Disclosure policy

The project maintainers will:

- review the report
- validate the issue
- coordinate fixes or specification updates when required

Responsible disclosure is encouraged to maintain the integrity of the governance framework.
