What is Janus?

Janus is a governance layer for AI-assisted systems.

It introduces a simple protocol that ensures actions performed with AI remain traceable, auditable, and human-validated.

Rather than replacing existing tools or workflows, Janus sits on top of them, providing a minimal structure that records decisions, preserves evidence, and allows system states to be rebuilt from verifiable events.

In simple terms

  • actions produce evidence
  • decisions remain human-validated
  • system state can be rebuilt from recorded events
