# Janus Core Lite — Status

Version: v0.1  
Status: Frozen  
Date: 2026

Janus Core Lite v0.1 has passed architectural coherence audit.

The core specification defines the minimal governance kernel for
auditable systems operating in AI-assisted environments.

The kernel includes:

- canonical append-only logs
- evidence model (E+ / E−)
- governance evaluation flows
- governance event model
- human authority boundary
- audit writer authority
- deterministic rebuildability

Supporting architecture documents define:

- governance pipeline
- governance scope model
- canonical governance event structure
- deterministic rebuild model

Future evolution of Janus must occur through additional RFCs.

Core semantics defined in Janus Core Lite v0.1 are considered stable.
